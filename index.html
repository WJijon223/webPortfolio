Fisherman Game

  • Python
  • Pygame

This 2D fishing game, developed using Python and the Pygame library, features engaging gameplay elements such as player movement, object spawning, and collision detection. Game components are structured using object-oriented programming, with modular classes that facilitate efficient development, easy maintenance, and potential for future expansion. The game's visual experience is enhanced through sprite animation, integrated seamlessly to create smooth, dynamic interactions between the player and game environment.
